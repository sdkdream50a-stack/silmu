# 공공계약법제 법령 정합성 감사 메모리

## 프로젝트 구조
- 토픽 데이터: `db/seeds/topics.rb` (초기 2개 토픽), `db/seeds/subtopics.rb` (서브토픽)
- 토픽 38~41은 프로덕션 DB에 rails runner로 직접 삽입 (시드 파일 미존재)
- 토픽 모델: `app/models/topic.rb` (law_content, decree_content, rule_content 컬럼)
- 토픽 뷰: `app/views/topics/show.html.erb` (캐스케이드 카드 레이아웃)
- 컨트롤러: `app/controllers/topics_controller.rb` (TOPIC_TOOLS, TOPIC_ICONS 매핑)

## 법령 계층 배치 규칙
| 섹션 | 배치 내용 | 위반 사례 |
|------|-----------|-----------|
| law_content | 법률 조항만 | 시행령 내용 금지 |
| decree_content | 시행령 (대통령령) | 법률/시행규칙 내용 금지 |
| rule_content | 시행규칙/예규/지침 | 법률/시행령 본문 금지 |

## 확인된 표준 금액 기준 (MEMORY.md 참조)
- 수의계약: 종합4억, 전문2억, 기타1.6억, 물품·용역 2천만원
- 1인 견적: 2천만원 이하 (특례 5천만원)
- 입찰공고기간: 10억미만 7일, 10~50억 15일, 50억+ 40일
- 적격심사: 물품·용역 2억원 이상
- 낙찰하한율: 89.745% (2026년 변경)

## 발견 패턴 (초회 감사)
