## 표준 금액 기준
- 수의계약: 종합4억, 전문2억, 기타1.6억, 물품·용역 2천만원
- 1인 견적: 2천만원 이하 (특례 5천만원)
- 적격심사: 2억원 이상 / 통과기준 95점
- 낙찰하한율: 89.745% (2026년~)
- 계약보증금: 계약금액의 10% / 면제: 3천만원 이하
- 지체상금: 1/1000 (종류별 차등 확인 필요) / 최고한도 10%
- 추가계약: 당초금액의 10% 이내
- 선금한도: 공사·제조 70%, 물품구매 30%

## 자주 발견되는 오류 패턴
- **검사기한 vs 대금지급기한 혼동**: 검사 14일 + 대금지급 5일 (총 19일)을 "14일 이내"로 잘못 기재하는 사례 반복
- **수의계약 금액 단순화 오류**: 공사별 세분류(종합4억/전문2억/기타1.6억)를 단일 금액으로 잘못 통합
- **1인 견적 기준 혼동**: 공통 2천만원(특례 5천만원)인데, 물품/용역을 더 낮은 금액으로 잘못 기재
- **특례 금액 사이트 내 불일치**: 플로차트(_private_contract_limit)에서 소기업5천/여성7천, 시드에서 소기업1억/여성1억 -> 수의계약 한도 특례 vs 1인견적 한도 특례 혼동
- **계약보증금 면제 기준 혼용**: _single_quote 플로차트 "5천만원", contract-guarantee-exemption 토픽 "3천만원" -> 시행령 제51조(계약서 생략) vs 제48조(보증금 면제) 구분 필요
- **국가계약법 수치 검증 없이 기재**: Gist 작성 시 지방계약법 수치를 국가계약법에도 유사 적용하는 오류 반복
- **입찰공고 기간 10일 vs 15일**: 중간 구간(1억~고시금액) 기간이 4차/6차에서 반복 확인 필요로 지적됨
- **입찰공고 조문번호**: 제35조 vs 제36조 혼용 (전부개정 확인 필요)
- **지체상금 요율 양법 차이**: 사이트(지방계약법) 1/1000 단일요율, Gist(국가계약법?) 공사1/2000·물품3/4000 차등요율 -> 양법 기준 혼재 주의
