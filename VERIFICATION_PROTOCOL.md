# 법령 현행화 검증 프로토콜

## 작성일: 2026-02-20

## 목적
사이트 전체 법령 내용을 2026년 2월 20일 기준 현행법과 일치시키기 위한 검증 및 수정 프로토콜

## 검증 기준 자료 (권위 있는 출처)
1. **법제처** (law.go.kr) - 법률, 시행령, 시행규칙 원문
2. **행정안전부** (mois.go.kr) - 입찰 및 계약집행기준 (예규)
3. **조달청** (pps.go.kr, g2b.go.kr) - 실무 지침

## 주요 법령 최신 버전 확인

### 1. 지방계약법 시행령
- 최신 개정: 2025년 3월 12일 (대통령령 제35382호)
- 시행일: 2026년 1월 2일

### 2. 지방자치단체 입찰 및 계약집행기준 (예규)
- 최신: 행정안전부 예규 제324호
- 시행일: 2025년 5월 1일

## 검증 완료 항목

### Phase 1: 법령 체계 정상화 ✅
- 30개 토픽의 law/decree/rule 섹션 재배치 완료
- law_content에서 시행령 제거
- decree_content에서 예규 제거

### Phase 2: 조문 인용 오류 수정 ✅
1. 수의계약 commentary
   - 분할계약: 제25조 → 제77조 (수정 완료)
   - 2인 이상 견적: 제25조 → 제30조 (수정 완료)

2. 수의시담 decree_content
   - 조항명: "수의계약의 방법" → "수의계약대상자의 선정절차 등" (수정 완료)
   - ③항 내용 정확하게 수정 완료

### Phase 3: 내부 일관성 검증 ✅
**금액 기준 일관성 확인 완료**:
- 일반 물품/용역: 2천만원 이하 1인 견적
- 특례 기업: 5천만원 이하 1인 견적
- 공사: 종합 4억, 전문 2억
- 모든 토픽 간 일관성 확인됨

## 검증 필요 (진행 중)

### Phase 4: 법령 원문 대조 검증

#### 우선 검증 대상 조문
1. **지방계약법 시행령 제25조** (수의계약 대상)
   - 금액 한도 정확성 확인
   - 특례 대상 기업 범위 확인

2. **지방계약법 시행령 제30조** (수의계약대상자의 선정절차)
   - 1인/2인 견적 기준 확인
   - 지정정보처리장치 이용 의무 확인

3. **지방계약법 시행령 제77조** (분할계약 금지)
   - 내용 정확성 확인

4. **입찰 및 계약집행기준 제69조, 제70조, 제74조**
   - 예규 최신 버전 내용 확인

#### 검증 방법
법제처, 행정안전부 공식 사이트에서 원문 확인 후 대조

**문제점**:
- WebFetch/WebSearch로 법령 페이지 접근 제한
- 법제처 사이트 구조상 직접 조문 추출 어려움

**대안**:
1. 법령 PDF 다운로드 후 직접 대조
2. easylaw.go.kr의 법령 해설 참조
3. 조달청 실무 가이드 참조

## 다음 단계

### 옵션 A: 수동 검증 (권장)
1. 법제처에서 주요 조문 PDF 다운로드
2. 토픽 내용과 일대일 대조
3. 불일치 발견 시 수정

### 옵션 B: 샘플 검증
1. 가장 중요한 10개 조문만 집중 검증
2. 오류 패턴 파악
3. 유사한 패턴이 있는 다른 토픽 일괄 수정

### 옵션 C: 전문가 의뢰
1. 공무원 또는 계약 전문가에게 검토 의뢰
2. 발견된 오류 목록 제공받아 수정

## 현재 상태 요약

- ✅ **법령 체계**: 100% 정상화 (36개 토픽)
- ✅ **조문 번호**: 발견된 오류 모두 수정
- ✅ **내부 일관성**: 금액 기준 등 일관성 확인 완료
- ⏳ **법령 원문 대조**: 진행 필요 (기술적 제약으로 수동 작업 필요)
