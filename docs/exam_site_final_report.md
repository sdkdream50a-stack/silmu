## Executive Summary

exam.silmu.kr은 **"잘 만든 도구 상자, 하지만 핵심 콘텐츠 공백"** 상태다.
CBT 시뮬레이션·플래시카드·스트릭 등 에듀테크 인프라는 박문각보다 앞서 있으나,
수험생이 가장 고통받는 "표준교재 요약"과 "실기 필답형 훈련"이 비어 있다.
시중 출판 교재가 전무한 지금이 이 공백을 선점할 유일한 창문이다.

## 1. 시장 개요

### 공공조달 시장 규모
| 연도 | 계약 규모 | 조달기업 수 |
|------|----------|-----------|
| 2014 | 111.5조원 | 293,418개 |
| 2023 | 208.5조원 | 572,118개 |

→ 9년간 87% 성장. 조달청이 자격증을 신설한 배경.

### 자격증 신설 타임라인
- **2025.03.04**: 국무회의 심의·의결 (국가기술자격법 시행령 개정)
- **2026년**: 첫 시험 시행 (수시검정, 연 2회 예정)
- **향후**: 조달청 자격 취득 우대 방안 마련 예정

### 확정 시험 일정
```
원서 접수: 2026.09.14~17
필기 시험: 2026.10.03  →  합격 발표: 10.12
실기 원서: 2026.10.12~15
실기 시험: 2026.11.14  →  최종 발표: 12.18
```
⚠️ 필기~실기 준비 기간: **단 32일**

## 3. 현재 시장의 블루오션 공백 3가지

### 🔵 공백 1: 표준교재 요약/해설 서비스 (가장 시급)
- 출판사 수험서: **전무** (2026.03 기준)
- 무료 PDF 1,000페이지+ → 독학자 패닉
- 브런치 개인 연재글에 수요 쏠림 → 제대로 된 서비스 없음
- **누가 먼저 챕터별 요약을 올리는 사이트가 되느냐** = 검색 트래픽 독점

### 🔵 공백 2: 실기 필답형 온라인 연습 (시장 내 유일)
- 필기 합격 후 **32일** 안에 서술/계산/빈칸/단답 실기
- 이를 온라인으로 타이핑 입력하며 채점해 주는 서비스: **어디에도 없음**
- 플래시카드 기반 입력 모드 = exam.silmu.kr만 가능

### 🔵 공백 3: 검색엔진 노출 오픈형 Q&A
- 조달파워TV 카페: 네이버 로그인 필수 → 구글 검색 미노출
- "공공조달관리사 OOO 뜻?" 검색 → 아무 결과 없음
- 사이트 내 문제별 댓글 → 구글 인덱싱 → 장기 SEO 자산

## 4. exam.silmu.kr 현황 평가

### ✅ 실제 강점 (경쟁사 대비)
1. **CBT 80문항 실전 시뮬레이션** — 박문각 빼고는 없는 기능
2. **플래시카드 모드** — 시장 내 유일
3. **오답 노트 + 북마크 + 분석 대시보드** — 학원도 없는 기능
4. **학습 스트릭** — 습관 형성 도구
5. **법령 가이드 68개 연계 자산** — 타 플랫폼 절대 불가
6. **무료** — 박문각 대비 진입 장벽 제로

### ❌ 치명적 약점 (즉시 보완 필요)
1. **강의 영상 없음** — 개념 학습 불가 (박문각, 조달파워TV에 독학자 뺏김)
2. **표준교재 요약 없음** — 핵심 Input 부재
3. **커뮤니티 없음** — 독학자 고립 → 조달파워TV 카페로 이탈
4. **실기 대비 콘텐츠 부족** — 4지선다만으로 필답형 대비 불가
5. **회원가입 유인 약함** — 비로그인 대부분 기능 접근 가능
6. **해설 법령 근거 미연결** — 왜 틀렸는지 근거 불명확

## 8. 최종 결론 — 솔직한 전문가 평가

### 지금 상태의 핵심 문제
> exam.silmu.kr은 수험생이 "문제 풀 준비가 됐을 때" 오는 사이트다.
> 그런데 수험생 80%는 "어디서 시작해야 할지 모르는" 상태다.

현재 사이트는 **Output(문제풀이) 중심**인데,
수험생에게 가장 필요한 건 **Input(개념 이해)**이다.

### 즉시 할 일 3가지 (오늘부터)

| 순위 | 작업 | 소요 시간 |
|------|------|---------|
| 1 | ExamQuestionComment UI 노출 | **반나절** |
| 2 | 모의고사 해설에 법령 가이드 링크 추가 | **하루** |
| 3 | 플래시카드 → 빈칸 타이핑 모드 추가 | **이틀** |

이 세 가지만 해도 현재 대비 **완전히 다른 사이트**가 된다.

### 6개월 후 목표
- 1회 기출 가장 빠른 복원 사이트 = 구글 검색 1위
- 합격자 데이터 → "exam.silmu.kr 이용자 합격률" 마케팅
- 2회 시험 수험생은 기출 + 에듀테크 도구 모두 있는 silmu.kr로
