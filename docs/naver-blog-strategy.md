# 네이버 블로그 연동 전략 (실무 silmu.kr)

## 목표
네이버 블로그를 검색 유입 채널로 활용하여 silmu.kr 트래픽 확보

## 전제
- 공무원 검색의 85%가 네이버
- 네이버 검색 상단: 블로그 > 카페 > 지식iN > 외부사이트
- silmu.kr 직접 SEO보다 네이버 블로그 우회 유입이 현실적

---

## 1. 블로그 세팅

**블로그명**: `실무노트` 또는 `계약실무 가이드`

**블로그 소개**:
```
현직 공무원을 위한 계약·예산 실무 정리
무료 자동화 도구: silmu.kr
```

**카테고리 구성**:
- 수의계약 실무
- 입찰·경쟁계약
- 예산·회계
- 감사사례 분석
- 서식·양식 모음
- 실무 도구 사용법

---

## 2. 콘텐츠 전략

### 절대 하면 안 되는 것
- ❌ 사이트 콘텐츠 그대로 복붙 (네이버 저품질 판정)
- ❌ "우리 사이트 좋아요 와주세요" 홍보글
- ❌ 하루에 5개씩 대량 발행

### 글 작성 공식: "문제 → 해설 → 도구 연결"

```
[제목] 공무원이 실제로 검색하는 문장 그대로

[본문]
1. 문제 상황 (공감) ← 3~4줄
2. 핵심 답변 ← 블로그에서 80% 해결해줌
3. 도구 연결 ← "더 정확한 판단은 여기서"
```
