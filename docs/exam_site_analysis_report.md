## 5. 즉시 개선 TOP 10 (우선순위 순)

### 🥇 1순위: 실기 필답형 대비 모드 추가
**왜**: 수험생 최대 공포 = 필기 합격 후 한 달 안에 실기
- 현재 플래시카드를 **"빈칸 채우기 + 단답 서술 모드"**로 확장
- 핵심 용어 → 정의 직접 입력 → 채점 (부분 점수 피드백)
- 실기 4가지 유형 분류: 서술형/계산형/빈칸/단답

### 🥈 2순위: 문제별 Q&A 댓글 활성화
**왜**: 조달파워TV 카페 대응 → 사이트 내부에서 질의응답 완결
- `ExamQuestionComment` 모델 이미 존재 → UI만 노출하면 됨
- 문제 해설 바로 아래 "이 문제 질문하기" → 학습 흐름 유지
- 댓글 달기 = 회원가입 필수 → 자연스러운 가입 유도

### 🥉 3순위: 핵심 3대 법령 집중 플래시카드 덱
**왜**: 모든 강사가 "국가계약법+시행령+조달사업법만 해라"라고 강조
- 기존 용어집에서 3대 법령 관련만 필터링한 "필수 덱"
- 홈/커리큘럼에서 "합격의 80%" 배너로 전면 배치

### 4. 모의고사 해설 → 법령 가이드 직접 링크
**왜**: 실무.kr 68개 법령 가이드는 타 플랫폼이 절대 가질 수 없는 자산
- 오답 해설에 `[관련 법령: 국가계약법 §00] → 실무.kr 가이드 바로보기` 링크
- 이 하나로 해설 품질이 박문각과 차별화됨

### 5. 표준교재 PDF 즉시 다운로드 배너
**왜**: 첫 방문자가 "어디서 시작하지?"일 때 즉시 도움
- 시험 정보 페이지와 홈에 "표준교재 무료 PDF 다운로드" 버튼
- 조달청/역량개발원 공식 링크로 연결

### 6. 맞춤형 60점 합격 진단 대시보드
**왜**: "나 지금 합격권인가?"를 수험생은 항상 궁금해함
- 모의고사 점수 기반 → 과목별 과락 위험 경고
- "이 속도로면 합격 가능성 OO%" 동적 표시

### 7. 회원가입 혜택 명확화
**왜**: 현재 비로그인 상태로도 대부분 기능 사용 가능 → 가입 유인 없음
- 로그인 전용: 오답 노트 저장 / 랭킹 / 학습 분석 / 챕터 진도 동기화
- "오늘 풀었는데 저장이 안 됐어요" 경험 → 가입 전환

### 8. 실전 CBT 타이머 + 시험 환경 재현
**왜**: CBT 방식 낯선 수험생이 많음, 실제 시험 환경 훈련 필요
- 80문항 120분 + 시험장 UI 재현
- 각 과목별 문항 수 잠금 (과목 간 이동 불가) → 실전 동일 조건

### 9. 일일 3문제 푸시 & 학습 목표 캘린더
**왜**: D-day까지 학습 루틴 유지가 합격의 핵심
- 매일 "오늘의 3문제" 배지
- 시험까지 남은 일수 × 하루 목표 문제 수 자동 계산

### 10. 학습 루틴 가이드 (주차별 플랜)
**왜**: "어디서부터 시작하지?"는 첫 방문자의 가장 큰 장벽
- "시험 6개월 전 시작 플랜 / 3개월 전 / 1개월 전" 상황별 로드맵
- 현재 정적인 전략 페이지를 동적 체크리스트로 전환

## 6. 전략적 차별화 방향

### 핵심 포지셔닝
> **"강의 들을 시간 없는 직장인을 위한 15분 인터랙티브 합격 도구"**

### 3가지 차별화 축

#### ① 법령 연계 해설 (경쟁사 불가)
```
문제 틀림 → 해설 읽기 → [실무.kr 법령 가이드 바로가기]
→ 조문 원문 + 실무 사례 + 감사사례 확인
→ 이 깊이는 학원도, 카페도 제공 불가
```

#### ② 실기-필기 통합 학습
```
플래시카드 → 앞면: 용어 / 뒷면: 정의 (필기용)
플래시카드 → 서술 모드: 정의를 직접 입력 (실기용)
CBT 모의고사 → 오답 발생 → 자동으로 실기 예상 주관식 연결
```

#### ③ 데이터 기반 맞춤 경로
```
모의고사 점수 분석 → "1과목 약함 → 용어집 1권 집중 추천"
스트릭 데이터 → "D-X일, 현재 속도로 OO 챕터 완료 가능"
랭킹 → "내 위치는 상위 OO%, 합격권 커트라인과의 거리 OO점"
```
