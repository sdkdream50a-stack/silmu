## 1. silmu.kr의 애드센스 강점 분석

| 강점 | 내용 | 수익 영향 |
|------|------|-----------|
| **타겟층** | 공무원/행정직, 30~50대, 안정적 소득층 | CPC 높은 광고 우선 배정 |
| **PC 비중 높음** | 모바일 대비 광고 단가 1.5~2배 | RPM 자동 상승 |
| **전문 법령 콘텐츠** | B2B·법률·행정 서비스 광고와 연관성 높음 | CPC 상위 카테고리 유치 |
| **계산기/도구 페이지** | 목적 지향적 방문 → 결과 후 광고 클릭 기회 | CTR 극대화 포인트 존재 |
| **체류시간 길다** | 도구 조작·결과 확인 과정에서 자연 체류 | 구글 품질 평가 유리 |

**예상 RPM**: 2,000~6,000원 (법령/행정 정보 + 계산기 도구 조합)

## 2. 광고 배치 최적화 전략

### 2-1. 모든 도구 페이지 공통 배치

```
┌─────────────────────────────────────────┐
│  [헤더 / 페이지 제목]                    │
│  ─────────────────────────────────────  │
│  [디스플레이 광고 — 도구 입력 상단]  ★  │
│  ─────────────────────────────────────  │
│  [도구 입력 영역]                        │
│              [사이드바 광고] ★★          │
│  [계산 결과 표시]                        │
│  ─────────────────────────────────────  │
│  [콘텐츠 내 광고 — 결과 바로 아래] ★★★ │  ← 클릭률 최고점
│  ─────────────────────────────────────  │
│  [관련 도구 내부 링크]                   │
└─────────────────────────────────────────┘
                         [앵커 광고 - 하단 고정] ★★★
```

- ★★★ **결과 영역 아래**: CTR 최고 (목적 달성 직후 = 주의 이완 = 클릭 가능성 최대)
- ★★★ **앵커 광고(하단 고정)**: 일반 광고 대비 CTR 3~5배, 반드시 활성화
- ★★ **사이드바(PC 전용)**: PC 방문자 비중 높은 silmu.kr에 유리
- ★ **도구 상단**: 노출수 극대화 (CTR 낮지만 브랜드 인지)

### 2-2. 법령 가이드 콘텐츠 페이지

```
[본문 첫 번째 단락 아래] → 디스플레이 광고
[스크롤 50% 지점]        → 콘텐츠 내 광고 (수익성 1위)
[본문 끝, 결론 위]       → 콘텐츠 내 광고
[앵커 광고]              → 항상 활성화
```

### 2-3. 도구별 연관 고단가 광고 카테고리

| 도구 페이지 | 예상 고단가 광고 카테고리 |
|-------------|--------------------------|
| 추정가격 계산기 | B2B 소프트웨어, 기업 금융, 보증보험 |
| 지체상금 계산기 | 법률 서비스(변호사/노무사), B2B 컨설팅, 기업보험 |
| 계약보증금 계산기 | 보증보험, 기업 금융, 법률 서비스 |
| 수의계약 판단 도구 | 법률/행정 서비스, 직무교육, B2B 소프트웨어 |
| 연가 계산기 | 금융(재테크), 보험, 의료, 부동산, 여행/레저 |

## 3. CPC / RPM 극대화 설정

### 반드시 실행할 3가지

1. **광고 균형 기능(Ad Balance) OFF**
   → AdSense 설정 > 광고 > 자동 광고 > 광고 균형 끄기
   → 켜두면 구글이 임의로 광고 송출을 줄여 수익 손실 발생

2. **광고 심사 센터 활성화 + 민감 카테고리 차단**
   → 성인·도박·저품질 광고 차단
   → 브랜드 광고 비율 증가 → CPC 상승

3. **앵커 광고 반드시 활성화**
   → 자동 광고 설정에서 앵커 광고(하단 고정) ON
   → CTR 3~5배 효과

### 트래픽 소스별 RPM (중요)

```
구글 검색 유입 > 네이버 검색 유입 > 직접 유입 > SNS 유입
```

→ **구글 SEO에 집중**하는 것이 RPM 상승의 핵심

## 4. 콘텐츠 전략 (CPC 높이는 글쓰기)

### 4-1. 고단가 키워드 타겟팅

롱테일 키워드로 의도를 명확히:

| 일반 키워드 | 롱테일 키워드 (전환율 높음) |
|-------------|---------------------------|
| 지체상금 계산 | 지체상금 계산기 직접 계산 |
| 수의계약 기준 | 2025 수의계약 금액 기준 계산 방법 |
| 추정가격 산정 | 용역 추정가격 계산기 원가항목 입력 |
| 연가일수 계산 | 공무원 연가일수 자동 계산 |

### 4-2. 체류시간 늘리는 글 구조

- **2,000자 이상** 작성 (구글 AI 품질 평가 기준)
- **목차(TOC) 삽입** → 페이지 내 이동 → 체류시간 증가
- **글 도입부**: 독자 문제 공감 + 해결책 예고 (이탈 방지)
- **내부 링크**: 관련 계산기/도구 간 촘촘하게 연결 → PV 증가

### 4-3. 고수익 콘텐츠 유형 (추가 발굴 대상)

1. 공공계약 관련 **비교/가이드**: "수의계약 vs 입찰, 언제 어떤 계약 방식을?"
2. **최신 법령 변경사항** 안내: 연도별 개정 내용 정리
3. **실무 사례 Q&A**: "지체상금 면책 사유 사례 5가지"
4. **계산기 활용 가이드**: 각 도구의 사용법 + 법적 근거 상세 설명

## 5. 기술적 최적화

| 항목 | 조치 | 효과 |
|------|------|------|
| **페이지 로딩 속도** | Core Web Vitals 최적화 | 광고 로딩 기회 증가 + 검색 순위 상승 |
| **메타 디스크립션** | 키워드 포함, 클릭 유도 문구 | 검색 CTR 향상 → 트래픽 증가 |
| **구조화 데이터(Schema)** | FAQ, HowTo 스키마 적용 | 리치 스니펫 → 클릭률 향상 |
| **A/B 테스트** | 광고 단위 이름 규칙화 | 위치별 CTR/CPC/RPM 비교 |
| **Analytics + Search Console** | 연동 후 페이지별 수익 분석 | 고수익 페이지 집중 투자 |
| **404/검색결과 페이지** | 광고 게재 제외 설정 | 무효 클릭 방지 + 품질 유지 |

## 6. 주의사항 & 실수 방지

### 절대 금지

- ❌ 본인 광고 클릭 (계정 정지)
- ❌ "광고를 클릭하세요" 등 클릭 유도 문구
- ❌ CTR 5% 초과 (비정상 클릭으로 의심)
- ❌ 광고와 콘텐츠 혼동 배치

### 신규 계정 특이사항

- 초기 3개월은 수익이 낮은 것이 정상 (구글 신뢰도 평가 기간)
- 꾸준한 고품질 콘텐츠 발행 + 자연 트래픽 증가가 핵심
- 빠른 수익보다 계정 신뢰도 축적이 장기적으로 유리

## 8. 핵심 플래시카드 (10문10답)

**Q1. 공무원 타겟이 애드센스에 유리한 이유?**
A. 경제적으로 안정적인 30대 이상 = 전환율 높음 = 높은 CPC 광고 우선 배정

**Q2. 계산기 도구에서 CTR이 가장 높은 위치?**
A. 결과 표시 영역 바로 아래 (목적 달성 직후 = 클릭 가능성 최대)

**Q3. PC 방문자 비중 높은 silmu.kr 최적 배치?**
A. 사이드바 광고 (PC 광고 단가 = 모바일의 1.5~2배)

**Q4. CPC 상승을 위해 차단할 광고 카테고리?**
A. 성인·도박 등 민감 카테고리 → 브랜드 광고 비율 증가 → CPC 상승

**Q5. 반드시 꺼야 하는 AdSense 설정?**
A. 광고 균형(Ad Balance) → OFF (켜두면 구글이 임의로 광고 줄임)

**Q6. CTR 3~5배 효과의 필수 광고 유형?**
A. 앵커 광고 (화면 하단 고정) → 도구 사이트에서 특히 효과적

**Q7. 체류시간을 늘리는 글쓰기 방법?**
A. 2,000자 이상 + 목차(TOC) 삽입 + 내부 링크

**Q8. 효과적인 롱테일 키워드 예시?**
A. "지체상금 계산 방법" → "지체상금 계산기 직접 계산" (의도 명확화)

**Q9. PV를 늘리는 구조 개선법?**
A. 계산기·도구·법령 가이드 간 내부 링크 촘촘히 연결

**Q10. RPM이 가장 높은 트래픽 소스?**
A. 구글 검색 유입 (네이버·SNS 대비 CPC·CTR 모두 높음) → 구글 SEO 집중
